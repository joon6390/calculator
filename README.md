# 리액트 훅이란?

- 함수형 컴포넌트에서 상태 관리와 생명 주기 기능 및 부가적인 기능을 활용할 수 있게 도와주는 새로운 기능임.

# useSate란?

- 함수형 컴포넌트에서 상태를 정의할 때 사용하는 리액트 훅임.

# useReducer란?

- 컴포넌트의 상태를 정의하는 또 다른 리액트 훅임.
- useSate 훅에 비하여 복잡한 상태관리를 할 때 효율적인 훅임.

# CSS-IN-JS란?

- 자바스크립트 파일 안에서 CSS 스타일을 정의하고 적용하는 접근 방식임.
- 자바스크립트로 개발하는 컴포넌트 기반 방식에 매우 적합함.
- 동적 스타일링을 쉽게 구현할 수 있음.
- s전역 스코프 문제, 상태 관리의 어려움, 유지보수의 복잡성을 해결함.

# Styled-Components란?

- CSS-IN-JS 기법 중 하나임.
- 스타일이 적용된 컴포넌트를 만들어서 스타일을 적용함.
- 외부 라이브러리이기 때문에 패키지를 설치해야만 사용 가능함.

# TailWind CSS란?

- 대표적인 Utility-First 기반의 CSS 라이브러리임.

# Utility-First란?

- 스타일을 미리 정의된 클래스 대신, 작은 스타일 속성을 유틸리티 클래스로 제공하여 개발자가 각 요소에 필요한 스타일을 조합해 나가는 방식임.
